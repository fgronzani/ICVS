ICVS — Atlas de Vulnerabilidade em Saúde
Mapa Metodologia

Sobre o Projeto

O ICVS é um atlas analítico que classifica os municípios brasileiros por vulnerabilidade em saúde, integrando dados de cinco sistemas públicos de informação.

Objetivo

Construir um índice composto que sintetize múltiplas dimensões de vulnerabilidade em saúde — desfechos, acesso a serviços e qualidade da atenção — para apoiar a tomada de decisão em saúde pública e o planejamento de ações em nível municipal.

O atlas permite a identificação de padrões regionais, a comparação entre municípios estruturalmente similares e o monitoramento temporal dos indicadores.

Abrangência

  • Cobertura geográfica: 5.570 municípios brasileiros
  • Período: Dados de 2021 (com capacidade para série histórica 2013–2023)
  • Indicadores: 15 indicadores em 3 dimensões
  • Fontes: SIM, SINASC, SIH/SUS, CNES (DATASUS) e IBGE

Arquitetura Técnica

Pipeline de Dados

O pipeline de processamento é implementado em Python e segue uma arquitetura modular:

  1. Coleta — Download automatizado via PySUS (SIM, SINASC, SIH, CNES) e API SIDRA/IBGE
  2. Cálculo de taxas — 15 indicadores per capita por município-ano
  3. Suavização bayesiana — Empirical Bayes (Gamma-Poisson) para municípios com população <10.000
  4. Normalização — Min-max robusta com truncamento nos percentis P5/P95
  5. Índice PCA — Análise de Componentes Principais para pesos data-driven por sub-índice
  6. Clustering — K-Means (k=6) para tipologia municipal
  7. Exportação — Geração de JSONs otimizados para o frontend

Frontend

  • Mapa coroplético — Leaflet.js com Canvas renderer e geometrias TopoJSON (5.3 MB)
  • Gráficos — Chart.js (evolução temporal, radar de perfil)
  • Hospedagem — Site estático (Cloudflare Pages)
