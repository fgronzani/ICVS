Sobre o Projeto

O que é o ICVS?

O ICVS — Índice Composto de Vulnerabilidade em Saúde é um atlas interativo que classifica os 5.570 municípios brasileiros por nível de vulnerabilidade em saúde. Ele integra dados de mortalidade, nascimentos, internações, infraestrutura e cobertura da atenção primária em um índice único, permitindo a identificação de territórios prioritários para políticas públicas de saúde.

Por que este atlas?

O Brasil possui uma enorme diversidade de condições de saúde entre seus municípios. Enquanto grandes centros urbanos concentram recursos, milhares de municípios pequenos enfrentam desafios como:

  • Alta mortalidade infantil e materna
  • Escassez de médicos e leitos hospitalares
  • Baixa cobertura da Estratégia Saúde da Família
  • Taxas elevadas de internações por condições evitáveis

O ICVS sintetiza essas dimensões em um índice comparável, transparente e replicável.

Dados Abertos

Este projeto utiliza exclusivamente dados públicos:

  • DATASUS — Sistemas SIM, SINASC, SIH, CNES (via pacote PySUS)
  • e-Gestor APS — Cobertura da Atenção Primária
  • IBGE — Estimativas populacionais e malha geográfica

Todo o código-fonte está disponível no GitHub sob licença aberta. O pipeline de dados é executado mensalmente via GitHub Actions, garantindo atualização contínua.
